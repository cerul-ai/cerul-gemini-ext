# Cerul Extension for Gemini CLI

**The video search layer for AI agents** — Gemini CLI extension.

Teach your AI agents to see. Search video by meaning — across speech, visuals, and on-screen text.

## Install

```bash
gemini extensions install https://github.com/cerul-ai/cerul-gemini-ext
```

You'll be prompted for your Cerul API key. Get one free at [cerul.ai/dashboard](https://cerul.ai/dashboard).

## What you can ask

Once installed, just ask Gemini naturally:

- "What did Sam Altman say about AGI timelines?"
- "Find Jensen Huang's comments on AI infrastructure at GTC"
- "Compare what Dario Amodei and Sam Altman said about open-source AI"
- "What did Andrej Karpathy explain about LLM training?"

## Tools

| Tool | Description |
|------|-------------|
| `cerul_search` | Search videos by topic, speaker, date, and source |
| `cerul_usage` | Check remaining API credits |
